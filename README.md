# zif

## Предназначение

Этот проект я создал для изучения React.

## Что это такое?

Это - форум, маленькая версия Двача.

## Структура

- Фронт - React
- Бэк - FastAPI

## Запуск

В данный момент, для запуска используется стандартный реактовый скрипт.

- Фронт - `npm run start`
- Бэк - `uvicorn --factory main:create_app`
